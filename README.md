# 💌 1st Anniversary Scroll Love Letter

첫 번째 기념일을 위해 만든 스크롤 기반 러브레터 웹사이트입니다.

흰 배경과 파스텔 핑크, 싱그러운 잎사귀와 꽃 장식을 사용해 청첩장 같은 분위기를 만들고, 사진과 짧은 편지 문장이 스크롤에 따라 하나씩 나타나도록 구성했습니다.

## 🌸 Concept

- 💗 1주년 기념 디지털 편지
- 🌷 청첩장 느낌의 밝고 따뜻한 디자인
- 📱 iPhone 17 기준 mobile-first 레이아웃
- ✍️ 손글씨 폰트 기반의 편지 같은 문장 스타일
- 🚪 오프닝 문 열림 애니메이션
- 📸 추억 카드, 사진 갤러리, 숨겨진 메시지
- 🎵 오프닝 버튼 클릭 후 배경 음악 재생 지원

## 🛠️ Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## 📁 Project

앱 코드는 `anniversary-letter/` 디렉토리에 있습니다.

```bash
cd anniversary-letter
npm install
npm run dev
```

## ✅ Build

```bash
npm run lint
npm run build
```
